## 스터디 규칙

  - 파일명은 `출처_문제번호.파일형식` 으로 생성

  - 개념+알고리즘+풀이 방식 자세하게 설명
  - 만약 상대가 이해 못하면 이해할 때까지 설명
  - 상대방이 설명한 문제는 반드시 본인의 스트릭에 추가할 것
